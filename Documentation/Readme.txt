===========  Description of the files   =======================


1) main_script.m  
    <Matlab script>: Containts the main routine to read images, find the stitched image, find differences

2) config_files/
    <Folder> : Contains the main config file "conf.ini"

3) images/
    <Folder> : Contains all the test images database

4) pyramid blend/
    <Folder> : Contains blending functions

5) Image mosaic using sift/
    <Folder> : Contains functions to find sift, ransac, homography and mosaics

6) inifile/
    <Folder> : Ini file code for reading and writing

7) highlight_blobs.m
    <Function> : Highlights the blobs that are passed and displays them
